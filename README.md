# Sistema Bancário

Projeto desenvolvido para praticar conceitos de Java Orientado a Objetos.

## Funcionalidades

- Criação de contas
- Depósito
- Saque
- Transferência
- Conta Corrente
- Conta Poupança
- Tratamento de exceções

## Conceitos utilizados

- Encapsulamento
- Herança
- Polimorfismo
- Classes Abstratas
- Exceptions
- Collections (ArrayList)
- Git e GitHub

## Tecnologias

- Java
- Eclipse

## Como executar

1. Clone o repositório
2. Abra no Eclipse
3. Execute a classe Main.java
